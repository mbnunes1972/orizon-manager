Perfil Ativo (sessao local)
Em producao na nuvem, o perfil sera definido por autenticacao de usuario. Localmente, selecione o perfil para testar os limites.
Limites de Desconto por Perfil
O desconto total (global + individuais) nao pode ultrapassar o limite do perfil ativo. Ao tentar exceder o limite, o sistema bloqueia e exibe aviso.
☁ Preparacao para Nuvem
• Os perfis aqui configurados serao migrados para um banco de dados de usuarios.
• Cada usuario tera um perfil associado via autenticacao (JWT/OAuth).
• O arquivo perfis_config.json e a fonte local — equivalente ao banco em nuvem.
• A funcao perfil_ativo_get() sera substituida por leitura do token do usuario.
• O limite e verificado no backend (Python) — nao apenas no frontend.
