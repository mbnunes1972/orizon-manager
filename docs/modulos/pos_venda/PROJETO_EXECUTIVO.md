## Visão geral

O projeto executivo é o detalhamento técnico do projeto de venda, desenvolvido após a medição. É o documento que orienta a produção e montagem.

---

## Etapa 14 — Validação da Planta de Pontos

**Responsável:** Projetista Executivo

**Ações:**
- Receber planta de pontos do cliente/arquiteto
- Validar compatibilidade com o projeto vendido
- Registrar aprovação ou solicitar ajustes

**Documentos:**

## User Stories

**US-PE-001** — Como projetista, quero registrar o recebimento da planta de pontos e validar sua compatibilidade.

**US-PE-002** — Como projetista, quero registrar a conclusão do projeto executivo e anexar o arquivo.

**US-PE-003** — Como conferente, quero comparar projeto de venda e executivo e registrar divergências.

**US-PE-004** — Como consultor, quero ser notificado quando o projeto executivo for aprovado pelo cliente.
